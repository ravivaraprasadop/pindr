Changelog

Every update, every improvement — the full history of Pindr.

v2.1 Latest March 2026
  • ✨New Ambient theme — deep navy, inspired by late-night journaling
  • ☁️Auto-sync to Google Drive every 30 minutes via background service worker
  • 🔒Token auto-refresh and 401 detection — stale sessions cleared gracefully
  • 📌Pin any board for instant popup access without opening the workspace
  • 🕐Live clock added to popup header
  • ⚙️New Settings modal with theme picker, data management, and sync controls
  • 🐛Fixed: checkboxes now persist state correctly across popup and workspace
  • 🐛Fixed: archive midnight trigger fires correctly on browser startup
  • 🐛Fixed: colour picker no longer closes immediately on click
  • 🐛Fixed: content no longer vanishes after refresh — blur + debounced save
v2.0 Stable January 2026
  • 🎨Complete UI overhaul — dotted journal aesthetic with 3 themes
  • ☁️Google Drive sync — optional backup directly to your personal Drive
  • 👤Google Sign-In with profile avatar display
  • 🗑️Trash bin — deleted boards held for 7 days
  • 📁Auto-archive — previous day's boards archived at midnight
  • 📤Export & Import — full JSON backup and restore
  • ⏱️Task timer with Chrome notifications
  • 🎨10 board colour themes: matcha, crimson, lavender, ocean, and more
  • 🏗️Migrated to Manifest v3 (required by Chrome)
v1.0 Original 2025
  • 🌱First release of Pindr
  • 📋Basic sticky board creation and editing
  • ✅Checkbox and bullet point task modes
  • 💾Local storage with chrome.storage.local
  • 🌙Light and dark themes
